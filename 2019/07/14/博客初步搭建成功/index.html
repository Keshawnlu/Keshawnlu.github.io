讲述一下这段时间自己坎坷的搭建过程吧，遇到最大的困难是图床，一开始用的是贴图网的图床，但是贴图网的免费版每七天就会删除相册里的图片，这谁顶得住呀，又不想花太多钱在
图床上，所以pass了贴图网，后来找了一些免费的图床，但是加载速度是真的不行(果然花钱的肯定比免费的好)，再后来又找到了七牛，本以为自己买的域名认证通过以后，绑定在七牛上，一切问题都迎刃而解，但是当我兴致勃勃去绑定域名的时候才发现这玩意还得备案，所以又去找备案的方法，填信息的时候发现需要一个备案服务号(好像叫这个，记不清了)，但是这东西好像得买了服务器以后才有，但我用的是github给自己的空间，淘宝上也搜不到这个服务号，所以不得不又放弃了七牛，正当我一筹莫展之时，突然发现了聚合图床，它独特的工作原理，使得加载图片速度也是非常的快(关键能免费用呀)，最终终于解决了图床的问题。

最后总结一下，就是用Github+Hexo+聚合图床初步搭完了这个博客，以后遇到的坑以后再慢慢解决慢慢讲吧…

初步用hexo+github搭建了一个博客网站，下一步就要继续完善网站，并考虑买一个域名

然后打算的话，会把一些杂七杂八的东西和编程相关的东西都放到这里来吧，以前CSDN的博客就让它先放在那里好了

加油油!!!

1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save

jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
